# Share file

Этот проект позволит вам поделится своими файлами, загрузив их в свой личный кабинет и скопировав ссылку на скачивание, которую можете скинуть кому угодно.

## Запуск

Чтобы запустить проект, после того как его склонируете себе на компьютер, проделайте следующие шаги:

- Создайте базу данных, к которой будет подключаться веб-приложение.
- Создайте рядом с файлом "example.properties" файл "application.properties", после чего заполните его из примера файла "example.properties".
- Запустите из корня проекта команду: ".\gradlew run" для запуска проекта.
- В браузере сделайте запрос по url адресу "localhost:8080/auth/register" и регистрируйтесь.
- Готово! Можете работать в веб-приложении.
